# Beta Consent Notice (단계 6.5.5 Day 1)

Butler 는 제품 품질 개선을 위해 사용자의 입력 패턴을 익명화하여 평가 데이터로 사용할 수 있습니다.
이 과정에서 이름, 회사명, 이메일, 전화번호, 문서명, 날짜, 경로, 서버 정보 등 식별 가능한 정보는
표준 토큰으로 치환됩니다.

원문 입력과 첨부 파일 원문은 평가 데이터셋에 저장하지 않습니다.
익명화 후에는 원문 복원이 불가능한 digest 값만 중복 방지 목적으로 보관합니다.

이 데이터는 intent 분류, deadline 판정, auto_apply 안전성 평가에만 사용되며,
6.5.6 이전에는 production candidate 판정이나 모델 학습에 사용하지 않습니다.
