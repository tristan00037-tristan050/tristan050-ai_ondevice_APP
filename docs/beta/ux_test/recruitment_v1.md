# UX 테스트 모집 가이드 v1

## 대상 선정 기준

**필수 조건:**
- 비개발자 (IT 직군 제외)
- macOS 노트북 사용자
- 주 3회 이상 문서 작업 수행자
- 만 20세 이상

**직무 분포 (5명):**
| 번호 | 직무 | 역할 |
|------|------|------|
| T1 | 영업 | 고객 제안서·메일 처리 多 |
| T2 | 마케팅 | 보고서·기획서 작성 多 |
| T3 | 회계/총무 | 거래내역·회계 분류 多 |
| T4 | 기획 | 내부 보고서·초안 작성 多 |
| T5 | 디자인 | 브리프·피드백 문서 처리 多 |

## 모집 방법

**내부 모집 (권장):**
- Slack/Teams 메시지: "Butler 베타 테스터 모집 — 30분, 소정의 보상"
- 부서장 경로로 직접 섭외 (5명 → 7명 예비 포함 확보 권장)

**모집 문구 (복사 사용):**
```
안녕하세요!

AI 문서 도우미 Butler 베타 테스터를 모집합니다.
개인정보는 회사 PC에서 처리되며 외부로 전혀 나가지 않습니다.

참여 방법:
- 소요 시간: 약 30분
- 장소: 본인 자리 또는 회의실
- 내용: 실제 업무 유사 시나리오 7개 체험
- 보상: 베타 1개월 무료 사용권 + 정식 출시 시 우선 도입 혜택

신청: [담당자 이름/연락처]까지 연락 주세요.
```
