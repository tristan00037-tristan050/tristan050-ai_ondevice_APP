# UX 테스트 시나리오 v1 — Butler PC Core 베타

**대상:** 비개발자 5명 (영업 1, 마케팅 1, 회계/총무 1, 기획 1, 디자인 1)  
**환경:** macOS 노트북 + Butler.app 설치 완료 상태  
**총 소요 시간:** 약 41분  
**진행 방식:** 동행 관찰 (Think-aloud 방식 — 행동 중 생각 소리 내어 말하기 요청)

---

## 시나리오 1 — 첫 실행 (5분)

**목표:** 설치·첫 실행 흐름의 직관성 확인

**절차:**
1. 테스터에게 `.dmg` 파일 전달 (USB 또는 공유 폴더)
2. `.dmg` 더블클릭 → Applications 폴더로 드래그 지시
3. Butler.app 실행 시도

**예상 막힘 지점:**
- "확인되지 않은 개발자" 보안 경고 팝업

**우회 안내 (관찰자가 이 시점에만 개입):**
- 시스템 설정 → 개인 정보 보호 및 보안 → "확인 없이 열기"
- 또는 Finder에서 Butler.app 우클릭 → 열기

**평가 항목:**
| 항목 | 평가 |
|------|------|
| 드래그 설치 직관성 | A(즉시) / B(잠깐 망설임) / C(도움 필요) |
| 보안 경고 대응 | A(스스로 해결) / B(힌트 후 해결) / C(도움 필요) |
| 첫 화면 도달까지 소요 시간 | ___분 ___초 |
| 막힌 지점 (자유 응답) | |

## 시나리오 2 — card_05 통장·거래내역 → 회계 분류 (10분)

**목표:** 핵심 카드 기능 체험 + 정확도 체감 측정

**사전 준비:** `docs/beta/samples/card_05_bank_to_accounting/input_bank.csv` 테스터 PC에 복사

**절차:**
1. Butler 화면에서 💰 카드 클릭
2. 파일 첨부 안내 확인 ("통장·거래내역 파일을 첨부해주세요")
3. 📎 버튼 클릭 → `input_bank.csv` 첨부
4. "이 거래내역 회계 분류해줘" 입력 → 전송
5. 진행률 표시 관찰
6. 결과 확인

**평가 항목:**
| 항목 | 평가 |
|------|------|
| 결과 정확도 체감 | A(정확) / B(대체로 맞음) / C(많이 틀림) |
| 처리 속도 체감 | A(빠름) / B(보통) / C(느림) |
| 진행률 표시 유용성 | A(유용) / B(있으나 마나) / C(혼란) |
| 결과 활용 의향 | 바로 쓸 수 있다 / 수정 후 쓸 수 있다 / 못 쓰겠다 |
| 소요 시간 | ___분 ___초 |

| 응답 품질 | A / B / C |

## 시나리오 5 — 외부 송신 0 검증 (3분)

**목표:** 로컬 처리 신뢰성 배지 발견 및 이해도 측정

**절차:**
1. 화면 상단 🔒 배지 발견 여부 관찰 (개입 없이 30초 대기)
2. 발견 못 하면 "화면 어딘가에 개인정보 보호 관련 표시가 있는데 찾아보세요" 힌트
3. 배지 클릭 → 상세 패널 확인
4. "Download Report" 클릭 → JSON 파일 확인
5. 시연자가 Wi-Fi 끄고 재전송 → 동일 작동 확인

**평가 항목:**
| 항목 | 평가 |
|------|------|
| 배지 자발적 발견 | 즉시 / 힌트 후 / 못 찾음 |
| "외부 송신 0" 의미 이해 | 즉시 이해 / 설명 후 이해 / 여전히 모름 |
| 신뢰도 변화 | 많이 높아짐 / 약간 높아짐 / 변화 없음 |
| Wi-Fi OFF 테스트 반응 (자유) | |

## 시나리오 7 — 종합 평가 (5분)

**목표:** 도입 의향 + 개선점 수집

**질문 목록:**

1. **회사 도입 의향 (5단계)**
   - 5: 당장 도입하고 싶다
   - 4: 테스트 후 도입 의향
   - 3: 관심은 있지만 망설임
   - 2: 아직은 아님
   - 1: 도입 생각 없음

2. **가장 좋은 점 (자유 응답, 3개 이내)**

3. **가장 아쉬운 점 (자유 응답, 3개 이내)**

4. **베타 기간(1개월) 실제 사용 의향**
   - 매일 사용할 것 같다
   - 주 1~2회 사용할 것 같다
   - 가끔 써볼 것 같다
   - 거의 안 쓸 것 같다

5. **이 앱이 없어진다면 (아쉬움 척도)**
   - 매우 아쉽다 / 아쉽다 / 그저 그렇다 / 상관없다

6. **추천 의향 (NPS: 0~10)**
   - 동료에게 이 앱을 추천할 가능성은? (0=전혀, 10=반드시)
