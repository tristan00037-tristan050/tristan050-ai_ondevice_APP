# card_03 — 기존 문서 기반 새 초안 샘플

**카드 목적:** 과거 작성 문서를 참고해 새 상황에 맞는 초안 생성  
**샘플 데이터:** 합성 가상 마케팅 보고서 (실제 데이터 아님)

## 입력 자료

- `input_past_doc.txt` — 과거 월간 마케팅 보고서 (2026년 3월)
- `input_new_situation.txt` — 새 상황 메모 (2026년 4월 지표)

## Butler 사용법

1. 📄 카드 클릭
2. 📎 버튼 → `input_past_doc.txt` 첨부
3. 📎 버튼 → `input_new_situation.txt` 추가 첨부
4. 다음 입력 후 전송:
   ```
   3월 보고서 형식 참고해서 4월 마케팅 성과 보고서 초안 써줘.
   수치는 새 상황 메모 파일 기준으로.
   ```

## 예상 결과 구조

```
# 2026년 4월 마케팅 성과 보고서 (초안)

## 1. 핵심 지표 요약
- 신규 유입: 1,420명 (전월 대비 +18%)
- 전환율: 3.8% (전월 대비 +0.6%p)
- 광고 ROI: 2.4x

## 2. 채널별 성과
...
## 3. 이슈 및 대응
...
## 4. 5월 계획
...
```

## 평가 기준

| 항목 | 기준 |
|------|------|
| 과거 문서 형식 유지 | 섹션 구조 동일한가 |
| 새 수치 반영 | 4월 지표가 정확히 들어갔는가 |
| 문체 일관성 | 3월 보고서와 비슷한 톤인가 |
| 초안 완성도 | 수정 없이 바로 쓸 수 있는가 |
