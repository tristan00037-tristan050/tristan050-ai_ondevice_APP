# card_02 — 남의 문서 → 우리 양식 샘플

**카드 목적:** 외부에서 받은 문서를 우리 회사 표준 양식으로 변환  
**샘플 데이터:** 합성 가상 제안서 (실제 데이터 아님)

## 입력 자료

- `input_external_doc.txt` — 외부 파트너 제안서 (가상)
- `input_our_template.txt` — 우리 회사 보고서 양식 (가상)

## Butler 사용법

1. 🔄 카드 클릭
2. 📎 → `input_external_doc.txt` 첨부
3. 📎 → `input_our_template.txt` 추가 첨부
4. 다음 입력 후 전송:
   ```
   외부 제안서 내용을 우리 회사 보고서 양식에 맞게 변환해줘.
   형식은 우리 템플릿 그대로, 내용만 외부 제안서에서 가져와.
   ```

## 예상 결과

우리 양식 헤더·섹션 구조를 유지하면서 외부 제안서의 핵심 내용이 채워진 문서

## 평가 기준

| 항목 | 기준 |
|------|------|
| 우리 양식 구조 유지 | 섹션 순서·형식 동일 |
| 외부 내용 누락 없음 | 주요 항목 빠짐없이 |
| 불필요한 외부 양식 잔재 없음 | 외부 회사 로고·표현 제거 |
