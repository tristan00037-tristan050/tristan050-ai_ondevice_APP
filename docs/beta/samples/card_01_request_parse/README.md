# card_01 — 요청 핵심 파악·정리 샘플

**카드 목적:** 받은 메일·메시지에서 핵심 요청을 구조화  
**샘플 데이터:** 합성 데이터 (실제 회사 자료 아님)

## 입력 자료

`input_email.txt` — 업무 이메일 (가상)

## Butler 사용법

1. 📥 카드 클릭
2. 📎 버튼 → `input_email.txt` 첨부
3. 다음 입력 후 전송:
   ```
   이 메일에서 무엇을 언제까지 어떤 자료로 처리해야 하는지 정리해줘.
   요청자, 기한, 필요 자료, 우선순위 형식으로.
   ```

## 예상 결과

```json
{
  "요청자": "홍길동 팀장 (영업3팀)",
  "핵심_요청": "Q2 실적 보고서 제출",
  "기한": "2026-05-15 오전 10시",
  "필요_자료": [
    "4월 매출 데이터 (ERP 추출)",
    "고객사별 계약 현황",
    "전분기 대비 증감율"
  ],
  "우선순위": "높음",
  "추가_요청사항": "임원 보고용 요약본 1장 별도 준비"
}
```

## 평가 기준

| 항목 | 기준 |
|------|------|
| 요청자 추출 | 정확히 추출됐는가 |
| 기한 파악 | 날짜·시간 올바른가 |
| 필요 자료 목록 | 빠진 항목 없는가 |
| 우선순위 판단 | 맥락에 맞는가 |
