# card_06 — 상대 양식에 우리 자료 채우기 샘플

**카드 목적:** 외부 기관 요청 양식에 우리 회사 자료를 자동 기입  
**샘플 데이터:** 합성 가상 입찰 신청서 (실제 데이터 아님)

## 입력 자료

- `input_blank_form.txt` — 빈 입찰 참가 신청서 양식 (가상 발주처)
- `input_our_data.txt` — 우리 회사 기본 정보 (가상)

## Butler 사용법

1. 📋 카드 클릭
2. 📎 → `input_blank_form.txt` 첨부
3. 📎 → `input_our_data.txt` 추가 첨부
4. 다음 입력 후 전송:
   ```
   빈 입찰 신청서 양식에 우리 회사 자료 채워줘.
   우리 자료 파일에서 맞는 정보를 찾아서 넣어줘.
   없는 항목은 [확인 필요]로 표시해줘.
   ```

## 예상 결과

```
입찰 참가 신청서

회사명: (주)예시테크
사업자등록번호: 123-45-67890
대표자: 김대표
주소: 서울시 강남구 테헤란로 123
...
[확인 필요]: 최근 3년 매출액 (재무제표 확인 필요)
```

## 평가 기준

| 항목 | 기준 |
|------|------|
| 양식 항목 매핑 정확도 | 우리 자료와 양식 항목 올바르게 연결 |
| 불확실 항목 표시 | [확인 필요] 빠짐없이 |
| 양식 원본 구조 유지 | 항목 순서 변경 없음 |
