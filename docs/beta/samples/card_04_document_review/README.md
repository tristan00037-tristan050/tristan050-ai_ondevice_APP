# card_04 — 첨부 문서 수정·보완 샘플

**카드 목적:** 작성 중인 문서의 빠진 내용·수정안을 AI가 제안  
**샘플 데이터:** 합성 가상 기획서 (실제 데이터 아님)

## 입력 자료

- `input_draft.txt` — 작성 중인 신규 서비스 기획서 (미완성)

## Butler 사용법

1. 📝 카드 클릭
2. 📎 → `input_draft.txt` 첨부
3. 다음 입력 후 전송:
   ```
   이 기획서에 빠진 부분이 있는지 검토하고,
   리스크 분석 섹션을 추가해줘.
   경쟁사 분석도 보완이 필요하면 채워줘.
   ```

## 예상 결과

```json
{
  "누락_항목": ["리스크 분석", "경쟁사 분석 상세", "수익 모델 근거"],
  "추가_제안": {
    "리스크_분석": "...",
    "경쟁사_분석": "..."
  },
  "수정_제안": [
    {"위치": "2.3 목표 시장", "현재": "...", "제안": "..."}
  ]
}
```

## 평가 기준

| 항목 | 기준 |
|------|------|
| 누락 항목 발견 | 실제 빠진 섹션 모두 지적 |
| 추가 내용 품질 | 맥락에 맞는 내용인가 |
| 수정 제안 적절성 | 원문 의도 훼손하지 않는가 |
