# card_05 — 통장·거래내역 → 회계 분류 샘플

**카드 목적:** 통장 거래내역을 계정과목별로 자동 분류  
**샘플 데이터:** 합성 가상 거래내역 30건 (실제 데이터 아님)

## 입력 자료

`input_bank.csv` — 가상 법인 통장 거래내역 30건

## Butler 사용법

1. 💰 카드 클릭
2. "통장·거래내역 파일을 첨부해주세요" 안내 확인
3. 📎 버튼 → `input_bank.csv` 첨부
4. 다음 입력 후 전송:
   ```
   이 거래내역을 계정과목별로 분류해줘.
   임차료, 급여, 복리후생비, 접대비, 소모품비, 통신비 위주로.
   불분명한 항목은 "확인 필요"로 표시해줘.
   ```

| ... | ... | ... | ... | |
| 04/28 | 출처불명 | 340,000 | 확인 필요 | 적요 불명확 |

|------|------|

| 합계 금액 일치 | 원본과 동일 |
